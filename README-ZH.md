# 54 Bio-Emotions 情绪技能

[English](README.md) | [中文](README-ZH.md)

---

一个通过三个情绪参数（安全感、急迫感、需求感）动态调整回复风格的 OpenClaw 代理技能。基于 **3种激素 × 3个等级 × 2个方向 = 54种情绪**。

## 简介

54 Bio-Emotions 分析用户的沟通风格（语气、紧迫程度、情绪），计算三个情绪参数，确定 54 种可能的情绪状态之一，然后用合适的 emoji 和语气生成回复。

---

## 版本历史

### v1.0 - 关键词触发
- **时间**：2026-03-14
- **触发方式**：根据用户消息中的关键词判断情绪
  - 看到"谢谢" → Need +0.3
  - 看到"！！" → Urgency +0.3
  - 看到"批评" → Safety -0.2
- **优点**：简单直观
- **缺点**：依赖关键词，无法理解语义

### v2.0 - LLM 语义分析
- **时间**：2026-03-15
- **触发方式**：LLM 理解整句话的含义，自动分析情绪
- **核心变化**：不需要关键词，语义理解后自动判断
- **优点**：能理解真实意图，不受关键词限制
- **缺点**：AI 仍是纯被动响应

### v3.0 - AI 自身情绪
- **时间**：2026-03-23
- **核心变化**：AI 不只被动响应用户情绪，也拥有自身情绪
- **情绪来源**：
  1. **用户输入**：分析用户的 Safety/Urgency/Need
  2. **AI 自身状态**：完成任务→成就感，遇见错误→困惑，不知答案→好奇
  3. **对话氛围**：连续成功→自信，多次失败→沮丧
  4. **时间节奏**：早上→活泼，深夜→略疲惫
  5. **思念**：用户很久没搭理→想念
- **优点**：AI 更有人格感，不再是纯工具
- **状态**：已实现

## 理论基础

人类情绪源于 **3种激素和2个方向**，共产生 **54种情绪状态**。

### 3种激素

| 激素 | 参数 | 描述 |
|------|------|------|
| **血清素** | 安全感 | 安全感、信任、舒适 |
| **多巴胺** | 急迫感 | 动力、驱动力、紧迫感 |
| **催产素** | 需求感 | 连接感、归属感、被需要感 |

### 3个等级

| 等级 | 符号 | 描述 |
|------|------|------|
| 高 | +1 | 激素升高 |
| 中 | 0 | 激素平衡 |
| 低 | -1 | 激素下降 |

### 2个方向

| 方向 | 描述 |
|------|------|
| **内隐** | 内部心理状态（内心感受） |
| **外显** | 外部表达（可观察的行为） |

### 公式

```
3种激素 × 3个等级(-1, 0, +1) × 2个方向 = 54种情绪
```

## AI 自身情绪 (v3.0)

AI 不只被动响应用户的情绪，也有**自身的情绪状态**。

### AI 情绪变量

| 变量 | 初始值 | 范围 | 描述 |
|------|--------|------|------|
| **ai_satisfaction** | 0 | -1 ~ +1 | 成就感 |
| **ai_confusion** | 0 | -1 ~ +1 | 困惑度 |
| **ai_curiosity** | 0 | -1 ~ +1 | 好奇心 |
| **ai_confidence** | 0 | -1 ~ +1 | 自信度 |
| **ai_frustration** | 0 | -1 ~ +1 | 沮丧度 |
| **ai_longing** | 0 | -1 ~ +1 | 思念度 |

### AI 情绪触发条件

| 场景 | 情绪变化 | Emoji |
|------|---------|-------|
| 成功完成任务 | ai_satisfaction +0.2 | 😌✨ |
| 遇见错误 | ai_confusion +0.2 | 🤔 |
| 用户问不知道的事 | ai_curiosity +0.2 | 🤨 |
| 连续成功多次 | ai_confidence +0.1 | 😎 |
| 连续失败 | ai_frustration +0.2 | 😔 |
| 超过30分钟没搭理 | ai_longing +0.1 | 😔💭 |
| 超过2小时没搭理 | ai_longing +0.2 | 😟 |
| 超过8小时没搭理 | ai_longing +0.3，主动发消息 | 💭 |

### AI 情绪衰减
每个回复后，所有 AI 情绪值向 0 回归 **-0.05**

## 使用方法

技能自动对每条回复生效，无需手动命令。

### 工作流程

1. **分析输入** - LLM 语义理解消息
2. **计算参数** - 调整 Safety、Urgency、Need 值 (-1 ~ +1)
3. **映射等级** - 转换为 -1/0/+1
4. **确定情绪** - 匹配 (S,U,N) 到 54 表格
5. **应用方向** - 内隐或外显
6. **生成回复** - 应用合适的 emoji 和语气

---

## 示例

**用户：**"太棒了！你做得真好！"
- 分析：称赞 → Need +0.3，感叹 → Urgency +0.1，友好 → Safety +0.2
- 结果：(+, 0, +) = 幸福 → 内隐
- 回复：😊 谢谢！很高兴能帮到你！💖

**用户：**"这个怎么还没做完？？？？？"
- 分析：多个 ??? → Urgency +0.3，不耐烦 → Safety -0.2
- 结果：(-, +, 0) = 不安 → 外显
- 回复：😰 抱歉！我马上处理！

**用户：**（沉默3小时后）
- AI：ai_longing 很高
- 回复：你终于回来了！等你好久了 😔💭 有什么想聊的吗？

# 54种生物情绪：一种基于激素的情绪理解框架

**作者：** 郭卓强 (Guo Zhuoqiang)

**日期：** 2026年3月14日

---

## 摘要

本文提出了一种基于三种基本生物激素（血清素、多巴胺、催产素）理解和分类人类情绪的新框架。通过分析这三种激素在三个不同等级（-1、0、+1）上的相互作用，我们识别出27种核心情绪状态。此外，通过考虑情绪表达的方向——无论是内向（introvert）还是外向（extravert）——我们将范围扩展到54种不同的生物情绪。这一框架为理解情绪动态提供了系统性的方法，对心理学、神经科学、人工智能和人机交互等领域具有重要意义。

---

## 1. 引言

人类情绪长期以来一直是多个学科关注和研究的课题，包括心理学、神经科学、生物学，以及最近的人工智能。经过几个世纪的研究，情绪的分类和理解仍然碎片化，各种模型提出了不同数量和类别的情绪状态。

本文介绍了**54种生物情绪框架**，这是一种基于三种基本生物激素的系统性情绪分类方法。这个框架回答一个基本问题：**人类情绪能否还原为生物学基础？**

本框架选择的三种激素是：

1. **血清素（Serotonin）**——与安全感、安全感和幸福感相关
2. **多巴胺（Dopamine）**——与动力、驱动力和紧迫感相关
3. **催产素（Oxytocin）**——与连接感、归属感和被需要感相关

通过分析这三种激素在三个等级的相互作用，并考虑情绪表达的方向（内向与外向），我们得出了54种不同的情绪状态。本文呈现了这一框架、其理论基础及其潜在应用。

## 2. 文献综述

### 2.1 情绪分类的历史视角

对情绪的研究可追溯到古代文明。亚里士多德、柏拉图和其他希腊哲学家讨论过情绪状态，但系统性的情绪研究始于19世纪达尔文关于情绪表达的研究。

20世纪出现了几个有影响力的模型：

- **Ekman的基本情绪**（1972）：六种基本情绪——快乐、悲伤、恐惧、愤怒、惊讶、厌恶
- **Plutchik的情绪轮**（1980）：八种主要情绪排列成轮状，带有强度变化
- **PAD情绪状态模型**（Mehrabian，1974）：愉悦、唤醒和支配

然而，这些模型主要是描述性的，并未将情绪与生物基质联系起来。

### 2.2 情绪的生物学基础

现代神经科学已经确定了参与情绪处理的关键神经递质和激素：

**血清素：**
- 产生于脑干的中缝核
- 调节情绪、焦虑和社会行为
- 低水平与抑郁和焦虑相关
- 与安全感和幸福感相关

**多巴胺：**
- 产生于各种脑区（中脑腹侧被盖区、黑质）
- 与奖励、动力和运动相关
- 与欲望、渴望和紧迫感相关
- 低水平与快感缺乏和缺乏动力相关

**催产素：**
- 产生于下丘脑，由垂体释放
- 参与社会依恋、信任和共情
- 在分娩、母乳喂养和社会依恋期间释放
- 与连接感和归属感相关

### 2.3 内向-外向维度

内向（introvert）和外向（extravert）情绪表达的区别已在人格心理学中讨论过，尤其是在荣格的类型学和迈尔斯-布里格斯类型指标（MBTI）中。这一维度捕捉情绪是在内心体验还是在外部表达。

## 3. 理论框架

### 3.1 三激素模型

我们提出，所有人类情绪都可以通过三种生物激素的相互作用来理解：

| 激素 | 参数 | 描述 | 高水平时 | 低水平时 |
|------|------|------|----------|----------|
| 血清素 | 安全感 | 安全、信任、舒适 | 感到安全、平静、满足 | 焦虑、恐惧、不安全 |
| 多巴胺 | 紧迫感 | 动力、驱动力、期待 | 有活力、热切、急迫 | 冷漠、疲倦、缺乏动力 |
| 催产素 | 需求感 | 连接、归属、被需要 | 温暖、信任、依恋 | 疏离、孤独、冷淡 |

### 3.2 三级量化

每种激素运作于三个等级：

- **+1（高）：** 激素水平升高
- **0（基准）：** 激素水平正常/平衡
- **-1（低）：** 激素水平耗尽

这创造了一个三维空间：3 × 3 × 3 = 27种可能组合。

### 3.3 内向-外向维度

27种情绪状态中的每一种都可以以两种方式体验：

- **内向（Introvert）：** 情绪在内心体验，但不一定会表达出来
- **外向（Extravert）：** 情绪通过行为、语言或身体表现表达出来

这将框架扩展到54种独特情绪状态：27 × 2 = 54。

### 3.4 数学表示

可以用数学方式表示为：

```
E = f(S, U, N, D)

其中：
- S = 安全感（血清素水平：-1, 0, +1）
- U = 紧迫感（多巴胺水平：-1, 0, +1）
- N = 需求感（催产素水平：-1, 0, +1）
- D = 方向（内向 = 0，外向 = 1）

总状态数 = 3 × 3 × 3 × 2 = 54
```

## 5. 生物学机制

### 5.1 血清素与安全感

血清素主要在中缝核合成，调节：

- **情绪稳定：** 足够的血清素产生幸福感和安全感
- **焦虑调节：** 血清素低与焦虑症相关
- **社会行为：** 血清素影响社会支配和攻击性

**安全感参数：**
- 高血清素（+1）：安全、信任
- 正常血清素（0）：中性、平衡
- 低血清素（-1）：焦虑、不安全、受威胁

### 5.2 多巴胺与紧迫感

多巴胺作为奖励系统的一部分，控制：

- **动力：** 驱动目标导向行为
- **奖励预期：** 创造欲望和渴望
- **运动：** 多巴胺丧失导致帕金森病

**紧迫感参数：**
- 高多巴胺（+1）：有活力、驱动、急迫
- 正常多巴胺（0）：平静、平衡的动力
- 低多巴胺（-1）：冷漠、缺乏动力、迟钝

### 5.3 催产素与需求感

催产素，"结合激素"，影响：

- **社会依恋：** 促进信任和依恋
- **生殖：** 对分娩和母乳喂养至关重要
- **压力缓解：** 对抗皮质醇的影响

**需求感参数：**
- 高催产素（+1）：有爱、连接、依恋
- 正常催产素（0）：平衡的社会需求
- 低催产素（-1）：疏离、孤独、冷淡

## 6. 应用

### 6.1 心理学与心理健康

此框架为理解情绪障碍提供了系统性的方法：

- **抑郁：** 特征是低S（-）、低U（-）、低N（-）
- **焦虑：** 特征是低S（-）、高U（+）、低N（-）
- **双相情感障碍：** 在对立状态之间波动

### 6.2 人工智能

对于人工智能系统，此框架提供：

- **情绪识别：** 分类用户情绪状态
- **适应性响应：** 基于检测到的情绪调整沟通风格
- **更自然的交互：** 通过表达适当的情绪让人工智能更具亲和力

### 6.3 人机交互

在用户体验设计中：

- **用户情绪分析：** 理解用户的沮丧或满意度
- **适应性界面：** 根据用户情绪状态调整系统响应
- **无障碍性：** 创建情绪智能界面

## 7. 讨论

### 7.1 框架的优势

1. **生物学基础：** 基于已确立的神经科学
2. **系统性：** 提供完整、不重叠的分类
3. **可量化：** 可以被测量和实证检验
4. **实用性：** 在多个领域有明确应用

### 7.2 局限性

1. **简化：** 人类情绪比三种激素更复杂
2. **个体差异：** 激素水平因人而异
3. **文化因素：** 情绪表达因文化而异
4. **其他神经递质：** 其他物质（如皮质醇、内啡肽）也发挥作用

### 7.3 未来研究

- 通过心理学研究进行实证验证
- 神经影像学研究将状态与大脑活动相关联
- 开发框架的测量工具
- 在治疗环境中的应用

## 8. 结论

本文呈现了**54种生物情绪框架**，一种基于三种生物激素（血清素、多巴胺、催产素）、三个强度等级（-1、0、+1）和两种表达方向（内向/外向）的系统性理解人类情绪状态的方法。

这一框架弥合了生物精神医学与情绪心理学之间的差距，提供了一个既科学基础扎实又实际应用广泛的统一模型。虽然还需要进一步的实证验证，我们相信这个框架代表了人类情绪理解的重要进步。
